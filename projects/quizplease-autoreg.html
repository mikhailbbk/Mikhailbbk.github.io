QuizPlease Autoreg

Автоматический мониторинг игр с Telegram уведомлениями и CI/CD

Python BeautifulSoup GitHub Actions Systemd Telegram API Linux

Описание проекта

Автоматизированная система для мониторинга расписания игр "Квиз, плиз! KLG". Система парсит сайт, анализирует доступность мест и отправляет уведомления в Telegram при появлении новых игр или изменении статуса.

Основные функции

  • Автоматический парсинг расписания игр каждые 30 минут
  • Фильтрация только классических игр "Квиз, плиз! KLG"
  • Определение доступности мест (свободно/резерв/нет мест)
  • Уведомления в Telegram о новых играх и изменениях статуса
  • Полный расклад по каждой игре в Telegram
  • Автоматический деплой через GitHub Actions
  • Запуск как системный сервис через Systemd
  • Логирование всех операций в файлы
  • Хранение истории игр для отслеживания изменений

Архитектура

Проект реализован на Python с использованием модульного подхода:

  • extract_classic_games.py - основной парсер и монитор
  • telegram_notifier.py - отправка уведомлений в Telegram
  • config.py - конфигурация (токены, настройки)
  • get_chat_id.py - утилита для получения Chat ID
  • data/ - хранение данных в JSON формате
  • logs/ - логи операций

CI/CD Pipeline

Настроен автоматический деплой через GitHub Actions при пуше в main ветку:

  1. Запуск workflow на Ubuntu
  2. Настройка SSH соединения с сервером
  3. Подключение к серверу по SSH
  4. Обновление кода из репозитория
  5. Установка зависимостей Python
  6. Перезагрузка системного сервиса

Технические детали

Особенности реализации:

  • Использование BeautifulSoup для парсинга HTML
  • Асинхронные HTTP запросы через requests
  • Хэширование игр для отслеживания изменений
  • Поддержка Markdown в Telegram сообщениях
  • Конфигурация через файлы для безопасности
  • Отказоустойчивость при ошибках сети
