Профессиональный профиль

Python разработчик с фокусом на backend разработку и DevOps инженерию. Специализируюсь на создании масштабируемых веб-приложений, парсинге данных и автоматизации рабочих процессов.

Цель: Ищу позицию Junior/Middle Python Backend Developer в команде, где смогу применять свои навыки и развиваться профессионально. Готов к решению сложных задач, изучению новых технологий и участию в интересных проектах.

Имею опыт полного цикла разработки — от проектирования архитектуры и написания кода до production-развертывания на Linux серверах и настройки CI/CD пайплайнов. Каждый проект проходит через все этапы: разработка, тестирование, деплой и мониторинг.

Дата начала разработки: 2024 год. Проекты в production: с 2025 года.

Backend Development

  • Python 3.11+, Django, Django REST Framework
  • Flask для легковесных и быстрых приложений
  • JWT аутентификация и системы безопасности
  • Парсинг данных (BeautifulSoup, Requests)
  • Проектирование и документирование REST API

Базы данных

  • PostgreSQL для production окружений
  • SQLite для разработки и тестирования
  • Создание миграций и проектирование схем
  • Оптимизация запросов и индексация
  • Стратегии резервного копирования

DevOps & Infrastructure

  • Администрирование Ubuntu Server
  • Настройка Nginx как reverse proxy
  • Развертывание на Gunicorn/Uvicorn
  • Управление сервисами через Systemd
  • CI/CD с GitHub Actions
  • Настройка безопасности и брандмауэров

Инструменты

  • Git, GitHub для контроля версий
  • Docker для контейнеризации
  • Bash-скрипты для автоматизации
  • Postman для тестирования API
  • VS Code как основная среда разработки
